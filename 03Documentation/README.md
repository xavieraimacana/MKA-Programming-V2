# Documentación del Proyecto

Este directorio contiene la documentación técnica y organizativa del sistema de gestión de la cafetería.

## Estructura del Repositorio

*   **[01Definition](file:///D:/POO/MKA-ProgrammingV2/01Definition):** Contiene la definición del problema y los objetivos del sistema.
*   **[02Requirements](file:///D:/POO/MKA-ProgrammingV2/02Requirements):** Contiene el estándar de requerimientos (SRS) y la matriz de características (Features).
*   **[03Documentation](file:///D:/POO/MKA-ProgrammingV2/03Documentation):** Manuales técnicos, guías de instalación de MongoDB en Java y documentación adicional de la API/código.
*   **[04UMLDiagrams](file:///D:/POO/MKA-ProgrammingV2/04UMLDiagrams):** Diagramas de clases, diagramas de casos de uso y diagramas de secuencia del sistema.
*   **[05UnitTests](file:///D:/POO/MKA-ProgrammingV2/05UnitTests):** Pruebas unitarias de JUnit para validar los managers, cálculos financieros de BigDecimal y el comportamiento de persistencia.
*   **[06Code](file:///D:/POO/MKA-ProgrammingV2/06Code):** Código fuente en Java estructurado bajo el patrón MVC.
*   **[07Other](file:///D:/POO/MKA-ProgrammingV2/07Other):** Archivos complementarios y evidencias multimedia de la ejecución del proyecto.
